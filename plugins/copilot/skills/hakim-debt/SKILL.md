# Hakim Debt

Search the explicit active scope for live `hakim:` markers. For each supported marker report file, line, shortcut, concrete ceiling, and upgrade trigger. Treat synthetic examples, archived phase records, and historical planning as non-live evidence unless the user explicitly asks to include them.

Default to read-only. Do not modify comments or ledgers while reporting debt.
