# Hakim Host Integrations

This directory contains maintained host projections and adapters for Hakim.

- `codex/`: Codex skills and repository-local integration files.
- `claude-code/`: Claude Code skills and hook integration.
- OpenCode integration is maintained in the repository's native plugin and
  packaging directories.
- GitHub Copilot uses `.github/copilot-instructions.md`.

Host-native trust, approval, sandboxing, activation, and removal controls remain
authoritative. A structural projection does not establish universal runtime or
model compatibility.
