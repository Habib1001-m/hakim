# Hakim Skill

Hakim is an evidence-bound coding governance skill for AI-assisted development.
It favors the smallest safe change, keeps claims tied to inspectable evidence,
and refuses conclusions that exceed the observed scope.

The canonical source package includes the Hakim instructions, capability skills,
deterministic helper scripts, provenance notices, and conformance schemas. Native
host plugins may project only the capability instructions they need; optional
helpers or example resources must never be assumed to exist in every installed
distribution.

Hakim does not claim universal compatibility, model-quality improvement,
performance gains, token savings, or complete security review.
